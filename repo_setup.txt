For: Team Members

Steps for setting up repository:

1. Fork this repository

2. Download free version of SourceTree from Atlassian

3. Open SourceTree and hit Clone/New

4. Put your forked repository url into Source Path

5. Choose a location on your computer where the repository will reside locally (ie. documents or desktop) and create a folder to store the repository (ie. desktop\CSCI3300Project)

6. Clone your forked repository to your computer

5. In order to added an upstream to this repository, follow the instructions in this link: http://stackoverflow.com/questions/13273852/how-do-i-update-my-forked-repo-using-sourcetree

8. In order to use the project with Unity, run Unity and select the project from your locally stored repository
